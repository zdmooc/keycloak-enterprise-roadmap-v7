# Runbook — Rollout Passkeys

## Objectif
Activer progressivement les passkeys.

## Étapes
1. choisir un périmètre pilote ;
2. vérifier la politique WebAuthn ;
3. documenter les parcours de secours ;
4. communiquer aux utilisateurs ;
5. mesurer le taux d’adoption ;
6. décider d’une extension ou d’un gel.
