# What to Adopt Now

## Adopter tôt si le besoin existe
- management interface propre ;
- metrics et health ;
- update compatibility ;
- client policies ;
- rotation des secrets ;
- passkeys sur périmètre pilote ;
- Organizations pour B2B structuré.

## Différer si la maturité interne manque
- extensions SPI lourdes ;
- topologies multi-cluster très ambitieuses ;
- customisation profonde sans stratégie d’upgrade.
