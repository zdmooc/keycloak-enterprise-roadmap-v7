# Backup, Restore, Export and Import

## Objectif
Distinguer clairement ce qui relève de la sauvegarde plateforme et ce qui relève de l’export logique.

## À retenir
- l’export/import de realm n’est pas une sauvegarde totale de plateforme ;
- la base de données reste un actif critique à sauvegarder et restaurer ;
- les secrets, manifests, policies et intégrations doivent aussi être versionnés ou sauvegardés.

## Preuves minimales
- date du dernier backup ;
- test de restore ;
- export logique de realm ;
- procédure documentée ;
- personne capable de la rejouer.
