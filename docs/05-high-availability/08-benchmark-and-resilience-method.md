# Benchmark and Resilience Method

## Objectif
Donner une méthode commune pour tester disponibilité et performance.

## Boucle recommandée
1. définir une hypothèse ;
2. exécuter la charge ;
3. injecter une panne ;
4. mesurer ;
5. comparer ;
6. corriger ;
7. rejouer.

## Sorties attendues
- scénario ;
- charge ;
- incident injecté ;
- comportement observé ;
- métriques ;
- conclusion ;
- décision.
