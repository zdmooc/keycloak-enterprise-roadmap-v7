# Operating Model and RACI

## Objectif
Éviter la confusion classique entre équipes sécurité, plateforme, IAM, applications et support.

## Rôles typiques
- équipe plateforme ;
- équipe IAM / Keycloak ;
- sécurité ;
- équipe DBA ;
- équipes applicatives ;
- exploitation / support.

## À clarifier
- qui crée un realm ;
- qui approuve un client ;
- qui gère les secrets ;
- qui fait la rotation ;
- qui déclenche une bascule PRA ;
- qui valide un upgrade.
