# 01 Iam Oidc Oauth2 Saml Basics

## Objectif
Poser les bases IAM et protocoles.

## Quand utiliser
Au début du parcours.

## Prérequis
- notions web et HTTP

## Concepts clés
- IAM
- authentication vs authorization
- OIDC
- OAuth 2.0
- SAML

## Procédure / design
Décrire les acteurs, les flux d’autorisation, les tokens, les claims et les cas d’usage typiques.

## Pièges fréquents
- confondre OAuth et OIDC
- croire qu’un access token remplace tous les contrôles applicatifs

## Checklist
- [ ] Les rôles des acteurs sont compris
- [ ] Les types de tokens sont distingués
