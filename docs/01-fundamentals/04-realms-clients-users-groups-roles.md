# 04 Realms Clients Users Groups Roles

## Objectif
Maîtriser les objets administratifs de base.

## Quand utiliser
Avant la modélisation d’un realm.

## Prérequis
- notions IAM simples

## Concepts clés
- realm
- client
- user
- group
- realm role
- client role

## Procédure / design
Illustrer les relations entre objets et les bonnes pratiques d’organisation.

## Pièges fréquents
- multiplier les rôles sans modèle
- utiliser les groupes comme unique mécanisme de sécurité

## Checklist
- [ ] Les objets sont distingués
- [ ] Le modèle cible est cohérent
