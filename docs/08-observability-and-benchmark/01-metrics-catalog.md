# Metrics Catalog

## Objectif
Lister les métriques utiles à la supervision, au troubleshooting et au capacity planning.

## Familles utiles
- disponibilité ;
- santé applicative ;
- erreurs ;
- latence ;
- JVM ;
- connexions DB ;
- activité d’authentification ;
- activité admin.

## Bonne pratique
Construire un petit noyau de métriques vraiment utiles avant de multiplier les graphes.
