# ADR-0001 - Repo scope

## Statut
Accepted

## Contexte
Le dépôt doit servir à la fois de guide d’apprentissage, de base d’architecture et de référentiel d’exploitation.

## Décision
Conserver une structure modulaire :
- docs
- labs
- manifests
- gitops
- runbooks
- architecture
- tools
- evidence

## Conséquences
- meilleure lisibilité ;
- itération possible par lots ;
- séparation claire entre théorie, exécution et preuves.
