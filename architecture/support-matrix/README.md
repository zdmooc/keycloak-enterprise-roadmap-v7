# Support Matrix

La matrice `ha-topologies.csv` ne remplace pas la documentation officielle.
Elle sert à classer les topologies internes du dépôt selon trois niveaux :
- **documented-supported-like** : proche des patterns officiellement documentés ;
- **context-validated** : crédible si ton contexte le valide ;
- **experimental** : à traiter avec prudence.

Toujours rapprocher cette matrice des guides officiels Keycloak et des tests réellement menés.
