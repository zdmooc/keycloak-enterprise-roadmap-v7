# tools/kcadm

Scripts opérationnels simples autour de `kcadm.sh`.

## Utilisation
- authentification admin ;
- création idempotente de realms ;
- création de clients ;
- exports.

## Bonnes pratiques
- ne pas stocker les secrets dans le dépôt ;
- rendre les scripts rejouables ;
- conserver les preuves dans `evidence/`.
