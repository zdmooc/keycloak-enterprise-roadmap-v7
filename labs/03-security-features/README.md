# Lab 03 — Security Features

## Objectif
Tester de manière isolée les briques de sécurité avancée.

## Scénarios
- token exchange ;
- passkeys / WebAuthn ;
- client policies ;
- rotation de secret ;
- service accounts.

## Méthode
Un scénario à la fois :
1. créer un realm de test dédié ;
2. appliquer la configuration ;
3. exécuter le parcours fonctionnel ;
4. observer logs, metrics et tokens ;
5. déposer la preuve.

## Modèle de fiche de scénario
- besoin ;
- design choisi ;
- configuration ;
- test exécuté ;
- résultat ;
- risques ;
- décision.
