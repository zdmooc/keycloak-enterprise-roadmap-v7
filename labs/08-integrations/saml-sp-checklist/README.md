# Lab — SAML SP Checklist

## But
Préparer une intégration SAML sans oublier les points qui cassent le plus souvent.

## Checklist
- [ ] métadonnées SP récupérées et versionnées
- [ ] ACS URL validée
- [ ] NameID choisi
- [ ] attributs/groups nécessaires identifiés
- [ ] stratégie de certificat / rollover documentée
- [ ] logout testé
