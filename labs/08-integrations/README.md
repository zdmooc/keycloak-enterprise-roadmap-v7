# Labs 08 — Integrations

Ce lot transforme la théorie en patterns rejouables.

## Contenu
- `spring-boot-resource-server/` : API Spring Boot validant des JWT émis par Keycloak.
- `spring-boot-login-client/` : appli web Spring Boot déclenchant un login OIDC.
- `spa-bff/` : pattern SPA + Backend For Frontend.
- `oauth2-proxy/` : protection d’une application amont via oauth2-proxy.
- `nginx-auth-request/` : exemple de frontal NGINX avec sous-requête d’authentification.
- `traefik-forward-auth/` : exemple de ForwardAuth avec headers propagés.
- `saml-sp-checklist/` : lot de validation pour intégration SAML.
- `k8s-service-account-auth/` : pattern machine-to-machine sur cluster.

## Résultat attendu
À la fin de ce lot, tu dois être capable de répondre proprement à la question :
**“pour cette application précise, quel pattern Keycloak est le plus adapté et pourquoi ?”**
